## Overview

In this project, goal was intially to scrap tiktok and allow bots to inject data on tiktok. But due to security reasons, now developing an applications which creates bots to update data and increase reach based on certian criteria.

## Key Features

- **Bot Creations**: Form to add new bots, and specific functionalities.
- **Customise Bots**: Customise bots to according to your needs.

## Technologies Used

- **React JS**: For building the front-end codebase with clean, reusable components.
- **GitHub Actions**: For automating the pipeline and syncing design changes to the repository.

## Challenges and Learnings

Main challenges here was to convert figma into reusable components and desings.
